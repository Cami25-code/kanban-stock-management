# PROJECT CONTEXT

## Project Name

Kanban Stock Management

---

## Vision

Kanban Stock Management est une application professionnelle de gestion de stock.

Le projet a deux objectifs :

1. Constituer un portfolio professionnel de niveau international.
2. Évoluer progressivement vers un SaaS destiné aux PME de l'UEMOA, puis au marché international.
